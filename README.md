## 介绍

- Gank.io 使用Kotlin语言实现的干货集中营客户端

## 项目模式
 - MVP
 - Kotlin
 - Retrofit
 - RxJava
 - RxAndroid
 - Material Design
 - Glide
 - ByeBurger
 - MultiStateView
 - base-rvadapter
 - lottie
 - circleimageview
